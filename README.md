
# (WIP) VELEST C Implementation

This repository contains a C implementation of the earthquake location code VELEST, originally written in FORTRAN 77. The implementation is based on VELEST version 3.1 (10th April 1995).

## Reference

Kissling, E., W.L. Ellsworth, D. Eberhart-Phillips, and U. Kradolfer: Initial reference models in local earthquake tomography, J. Geophys. Res., 99, 19635-19646, 1994.
